Summary: Latitude vs. X

The purpose of the WeatherApp project was to compare various weather conditions (i.e. temperature, humidity, cloudiness, and wind speed) between the northern and southern hemispheres to determine if they were different. To do this, a data set of 500 cities was randomly generated using Longitude and Latitude coordinates. Weather data was then pulled from OpenWeatherMap API for analysis.

After compiling the data set, I used Matplotlib to generate scatterplots comparing the temperature, humidity cloudiness, and wind speed for cities in each hemisphere. The source code and visualizations can be found on this site.
